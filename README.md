# Web_Integration
# Projet DiagExpress - Réserver un Entretien

Ce projet est une plateforme permettant aux utilisateurs de réserver un entretien pour la maintenance de leurs véhicules. Il inclut une interface utilisateur réactive permettant de sélectionner des services, une date, et de soumettre des informations personnelles.

## Instructions pour exécuter le projet

### Prérequis

Avant de démarrer le projet, assurez-vous d'avoir les éléments suivants installés sur votre machine :
- **Node.js** (version >= 14)
- **npm** (Node Package Manager)

### Étapes pour exécuter le projet

1. **Cloner le dépôt :**

   Si ce n'est pas déjà fait, clonez le dépôt du projet sur votre machine locale :

   ```bash
   git clone https://github.com/MedEzzedineBenZbiba/Web_Integration.git

2. **Installer les dépendances :**
- Allez dans le dossier du projet avec la commande suivante : "cd Web_Integration"
- Ensuite, installez les dépendances avec la commande : "npm install"

3. **Compiler le fichier CSS :**
Utilisez la commande suivante pour générer le fichier CSS "npm run build"

4. **Et voilà !: **
Ouvrez les interfaces avec votre navigateur préféré en ouvrant directement le fichier HTML ou en démarrant un serveur local pour visualiser l'application dans votre navigateur.
